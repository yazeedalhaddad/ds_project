What this project does

This tool uses a machine learning model trained on thousands of simulated bus trips. It understands the relationship between weather, time of day, and bus routes to provide a delay estimate in seconds.

How It Works

  1. Input: You select a bus route, start stop, destination, and departure time.
  2. Weather Data: You provide the expected weather conditions for the trip (e.g., temperature, wind, rain/snow).
  3. AI Model: The browser sends this information to our Python Flask server, which passes it to the pre-trained CatBoost model.
  4. Delay Estimate: The model returns a predicted delay in seconds. Positive values mean the bus is late; negative values mean it's early. This is displayed in the result box.
